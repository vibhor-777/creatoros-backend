# CreatorOS Backend

CreatorOS is a Node.js + Express.js backend for a student-only digital marketplace built for India. The platform has 6 phases:

- **Phase 1** — Digital Vault (students sell PDFs, notes, code templates)
- **Phase 2** — Creator Bazaar (verified student freelance services)
- **Phase 3** — AI SaaS Engine (AI listing optimizer, business architect)
- **Phase 4** — Dev Bounty Zone (bug bounties, GitHub-verified tech gigs)
- **Phase 5** — NeoBank Wallet (RazorpayX virtual accounts)
- **Phase 6** — Physical Goods Engine (Shiprocket escrow logistics)

## Getting Started

1. Clone the repository
2. Run `npm install`
3. Copy `.env.example` to `.env` and fill in your configuration
4. Run `npm run dev` to start the development server
